AI Sight

AI Sight is an Android app for Offline Object Recognition!
It can recognize objects belonging to 1000 categories!

Privacy Policy

AI Sight requires the following permissions:
- Camera Access
- Storage Access

The app needs camera access for the user to click pictures of objects to be recognized, and needs storage access for the user to be able to select an image from the phone's gallery

The app works offline: user data is neither collected nor shared with third party services

Therefore, AI Sight does not collect/store/distribute any personally identifiable information
